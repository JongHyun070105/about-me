# 📱 대충톡 (DaechungTalk)
**"바쁠 땐 대충, 하지만 똑똑하게"**  
안드로이드 전용 온디바이스 AI 기반 카카오톡 자동 답장 제안 서비스입니다.

---

## 🌟 주요 특징
- **🔒 개인정보 보호 최우선**: Google의 온디바이스 모델인 **Gemini Nano**를 우선적으로 사용하여 사용자의 대화가 기기 밖으로 나가지 않도록 설계되었습니다. (지원 기기 및 상황에 따라 Cloud Fallback 지원)
- **📅 스마트 일정 연동 (RAG)**: 사용자의 시스템 캘린더(Google, 삼성 등)와 연동하여 "오늘 뭐 해?", "내일 시간 돼?" 같은 질문에 실제 일정을 바탕으로 대답합니다.
- **💬 대화 맥락 파악**: 단순히 키워드에 반응하는 것이 아니라, 해당 채팅방의 최근 대화 이력을 분석하여 자연스러운 말투로 답장을 제안합니다.
- **🛡️ PII 마스킹**: 전화번호, 계좌번호, 주민번호 등 민감한 개인정보는 AI에게 전달되기 전 자동으로 마스킹 처리되어 안전합니다.
- **🎯 채팅방 맞춤 설정**: 방마다 '자동 답장 활성화' 여부와 '특별 대화 규칙'을 직접 설정할 수 있습니다.

---

## 🛠 주요 기능
- **메시지 캡처**: Accessibility Service와 Notification Listener를 통해 카카오톡 알림 및 메시지를 실시간으로 파악합니다.
- **플로팅 답장 UI**: 상대방의 메시지가 오면 화면에 나타나는 3가지 AI 추천 답장 중 하나를 골라 즉시 답장을 보낼 수 있습니다.
- **캘린더 RAG**: 전후 1개월간의 일정을 분석하여 시간대별, 날짜별로 정리된 정보를 AI에게 주입합니다.

---

## 🏗 기술 스택
- **Frontend**: Flutter
- **Native (Android)**: Kotlin (AccessibilityService, NotificationListenerService, RemoteInput API)
- **AI Engine**: Google AICore (Gemini Nano), GenAI SDK (Cloud Fallback)
- **Database**: SQLite (채팅 로그 및 설정 저장)
- **Language**: Kotlin, Dart

---

## 🚀 시작하기
1. 앱 설치 후 실행합니다.
2. **필수 권한**을 차례대로 허용합니다:
   - **접근성 권한**: 카카오톡 메시지 캡처를 위해 필요합니다.
   - **알림 접근 권한**: 메시지 수신 및 답장 전송을 위해 필요합니다.
   - **다른 앱 위에 표시**: 플로팅 답장 UI를 위해 필요합니다.
   - **캘린더 권한**: 일정 기반 답변 생성을 위해 필요합니다.
3. 답장을 원하는 채팅방 설정에서 **'자동 답장 활성화'**를 켭니다.
